# 🏔️ Mount Jungle - Roblox Parkour Game

A comprehensive jungle-themed parkour game for Roblox with multiplayer support, multiple levels, shop system, and leaderboards.

## Features

✅ **Parkour Mechanics**
- Jump, climb, wall run, vine swing
- Smooth movement system
- Parkour-specific animations

✅ **Jungle Maps** (5 Levels)
- Level 1: Beginner - Basic jumps & climbs
- Level 2: Intermediate - Vine swinging
- Level 3: Advanced - Complex obstacles
- Level 4: Expert - Speed challenges
- Level 5: Master - Everything combined

✅ **Shop System**
- Character skins & cosmetics
- Power-ups (Speed Boost, Double Jump, etc)
- In-game currency system

✅ **Multiplayer**
- Real-time leaderboards
- Player rankings
- Global scoring

✅ **UI Systems**
- Main menu
- In-game HUD
- Settings & controls
- Shop interface
